# React Campo Minado

Jogo do campo minado em React com cronometro.

![Screenshot](campoMinado.gif)

## Tecnologia
- React
- Stopwatch
- Dialog
- Redux

## Intalação

```sh
$ git clone https://github.com/ldonizete/reactjs-campo-minado.git
$ cd react-minesweeper
$ yarn install
$ yarn start
```
